CommandLine (DOS)
===========

Как вывести информацию устройства в сети?
-----------------------------------------
```
> ipconfig
```

Как вывести IPv6 информацию?
----------------------------
```
> ipv6config /all
```

Как узнать IP по домену?
------------------------
```
> nslookup b2server.pt.pka
```

Как узнать путь до узла?
------------------------
```
> tracert b2server.pt.pka
```
